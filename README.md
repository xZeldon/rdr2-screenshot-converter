# RDR2-Converter

Простой скрипт для конвертации скриншотов из игры Red Dead Redemtion 2 во всеми привычный формат JPEG.

## Использование

Устанавливаем зависимости и запускаем:

```bash
yarn install
yarn start
```

Чтобы скомпилировать в исполняемый файл (windows only):
``` bash
yarn compile
```
